# NPM Registry @LarCity

Simple, hosted NPM registry implemented with Verdaccio

## File Structure

```shell
.
├── README.md
├── app
│   ├── conf
│   │   └── config.yaml # See Example: https://github.com/verdaccio/docker-examples/blob/master/docker-local-storage-volume/conf/config.yaml
│   ├── plugins
│   └── storage
│       └── @larcity
└── docker-compose.yml
```
